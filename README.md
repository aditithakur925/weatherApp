# SkyCast

A modern, simple, and intuitive mobile application to check real-time weather conditions and forecasts. Built using Java for Android, this application aims to provide users with accurate weather information for any location worldwide.

## ‚ú® Features

* **Real-time Weather Data:** Get up-to-the-minute weather conditions for the current location or any searched city.
* **7-Day Forecast:** View a detailed weekly weather forecast including temperature highs and lows.
* **Hourly Forecast:** Check hourly temperature and condition changes for the next 24 hours.
* **Search Functionality:** Easily search for weather in any city using the search bar.
* **Detailed Information:** Display key metrics like humidity, wind speed, pressure, and 'feels like' temperature.
* **Intuitive UI:** A clean and easy-to-read user interface.

## üöÄ Getting Started

Follow these instructions to set up the project locally.

### Prerequisites

* Android Studio
* Android SDK (v21 or higher)
* A working API key for your chosen weather service (e.g., OpenWeatherMap).

### Installation

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/abhirai7/SkyCast.git](https://github.com/abhirai7/SkyCast.git)
    ```

2.  **Open in Android Studio:**
    Open the cloned directory as an existing Android Studio project.

3.  **API Key Configuration:**
    You need to include your Weather API key. Create a file named `local.properties` in the root directory of the project (if it doesn't exist) and add your key:

    ```properties
    # local.properties (in the root of the project)
    WEATHER_API_KEY="YOUR_API_KEY_HERE"
    ```
    *Note: The specific location for the key may vary based on your implementation.*

4.  **Sync and Run:**
    * Let Gradle sync the project dependencies.
    * Select a target device or emulator.
    * Click the **Run** button to build and install the app.
